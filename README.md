# My First API

## 🇧🇷 Português

API REST desenvolvida com Spring Boot com o objetivo de praticar conceitos fundamentais de desenvolvimento backend, incluindo criação de endpoints, organização em camadas e boas práticas.

---

## 🚀 Funcionalidades

- Criação de endpoints REST
- Operações básicas (CRUD)
- Estrutura organizada em camadas (Controller, Service, Repository)
- Integração com banco de dados (se aplicável)

---

## 🛠 Tecnologias utilizadas

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven

---

## 📂 Estrutura do projeto
```
src/
├── controller/
├── service/
├── repository/
└── model/
```

---

## ▶️ Como executar o projeto

### Pré-requisitos:
- Java 17+
- Maven

### Passos:

```bash
# Clonar repositório
git clone https://github.com/CaiorrRamos/my-first-api.git

# Entrar na pasta
cd my-first-api

# Rodar aplicação
mvn spring-boot:run
````
| Método | Endpoint | Descrição      |
| ------ | -------- | -------------- |
| GET    | /api/... | Listar dados   |
| POST   | /api/... | Criar registro |
| PUT    | /api/... | Atualizar      |
| DELETE | /api/... | Remover        |
